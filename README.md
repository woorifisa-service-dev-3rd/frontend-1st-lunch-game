## 팀 Commit Rule

### Message Head
- 처음 파일을 만드는 경우 [INIT]
- 처음 파일을 작성하는 경우 [ADD]
- 기능을 수정하거나 바꾸는 경우 [UPDATE]
- 버그를 수정한 경우 [FIX]

### Message Body
- 커밋 메시지는 영어로 진행
- 커밋 메시지의 첫 단어의 첫 글짜는 대문자로 시작
- 커밋 메시지의 띄어쓰기는 _로 변경
